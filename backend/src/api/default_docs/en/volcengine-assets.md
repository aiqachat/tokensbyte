# Volcengine Ark Material Library API Integration Guide

This gateway is deeply compatible with the material library and asset management capabilities of Volcengine Ark. Developers can directly invoke Volcengine's native material and asset group management APIs via the gateway's `/api` proxy endpoint, and also manage their uploaded assets securely with user-level isolation.

### Official Documentation
For complete parameters, limits, and the latest capabilities, please refer to the Volcengine Ark official docs:
* [Material Library Official Docs](https://console.volcengine.com/ark/region:cn-beijing/docs/82379/2333565?lang=zh)

---

## 1. Basic Request Information

For high efficiency and security, all material APIs are submitted via `POST` requests, communicating with the gateway using JSON format.

* **Base URL**: `https://{{domain}}/api`
* **HTTP Method**: `POST`
* **Required HTTP Headers**:
  * `Content-Type: application/json`
  * `Authorization: Bearer sk-your_api_key_value_here` *(Your API key generated in the dashboard)*

---

## 2. Common Query Parameters

When calling any Volcengine Ark asset API, specify the `Action` and `Version` in the query parameters of the URL:

| Parameter | Required | Type | Description |
| :--- | :--- | :--- | :--- |
| `Action` | Yes | `string` | The asset action to perform (e.g., `CreateAsset`). See the core API list below. |
| `Version` | No | `string` | The API version, defaults to `2024-01-01`. |
| `ns` | No | `string` | Plugin namespace, options: `asset_manager` (domestic version, default) or `asset_manager_intl` (international version). |

*Example Request URL*:
`POST https://{{domain}}/api?Action=CreateAsset&Version=2024-01-01&ns=asset_manager`

## 3. Core API Endpoints

### ① Register/Create Asset (CreateAsset)
Registers a media file URL stored in an object storage (e.g., TOS) as a Volcengine Ark asset.
* **URL Action**: `CreateAsset`
* **JSON Request Body**:
```json
{
  "AssetType": "video", // Asset type: "image" or "video"
  "URL": "https://your-tos-bucket.tos-cn-beijing.volces.com/path/video.mp4", // Publicly accessible URL of the file
  "Name": "test_video_asset", // Asset name (up to 100 characters)
  "Description": "This is a background video asset" // Optional, asset description
}
```
* **JSON Response**:
```json
{
  "ResponseMetadata": {
    "RequestId": "20260714101500...",
    "Action": "CreateAsset",
    "Version": "2024-01-01",
    "Service": "ark",
    "Region": "cn-beijing"
  },
  "Result": {
    "Id": "asset-20260714...", // Global unique asset ID generated by Volcengine
    "Status": "active" // Status: active, processing, failed
  }
}
```
* **Gateway Enhancement & Security**:
  1. **Project Override**: The gateway automatically overrides `ProjectName` with the system-configured project name to ensure unified asset management.
  2. **Local Ownership Binding**: Upon successful creation, the gateway automatically binds the asset to your account in the local database. You can view and manage it on the "My Assets" page.

---

### ② Get Asset Info (GetAsset)
Retrieves the detailed status and metadata of a single asset.
* **URL Action**: `GetAsset`
* **JSON Request Body**:
```json
{
  "Id": "asset-20260714..." // The asset ID
}
```
* **Notes**: The gateway enforces strict data isolation. Non-admin users can only query assets they created, otherwise a `403 Forbidden` error is returned.

---

### ③ Delete Asset (DeleteAsset)
Deletes the asset from Volcengine Ark and removes the local database ownership record.
* **URL Action**: `DeleteAsset`
* **JSON Request Body**:
```json
{
  "Id": "asset-20260714..." // The asset ID to delete
}
```
* **JSON Response**:
```json
{
  "ResponseMetadata": {
    "RequestId": "20260714102000...",
    "Action": "DeleteAsset",
    "Version": "2024-01-01",
    "Service": "ark",
    "Region": "cn-beijing"
  },
  "Result": {}
}
```
* **Notes**: Non-admins can only delete their own assets. The gateway verifies ownership, calls Volcengine Ark API to delete the cloud asset, and then clears the local record.

---

### ④ Create Asset Group (CreateAssetGroup)
Volcengine video generation models (e.g. Seedance) support multi-image reference and subject reference. You can group assets of the same type together.
* **URL Action**: `CreateAssetGroup`
* **JSON Request Body**:
```json
{
  "Name": "My Reference Group", // Group name
  "Description": "Reference portraits for video generation" // Optional description
}
```
* **JSON Response**:
```json
{
  "ResponseMetadata": {
    "RequestId": "20260714103000...",
    "Action": "CreateAssetGroup",
    "Version": "2024-01-01",
    "Service": "ark",
    "Region": "cn-beijing"
  },
  "Result": {
    "Id": "group-20260714..." // Global unique asset group ID generated by Volcengine
  }
}
```
* **Notes**: Created asset groups are also isolated and bound to the creator's account.

---

### ⑤ Other Supported Actions
The gateway transparently proxies and supports the following Volcengine Ark asset actions:
* `UpdateAsset` (Updates name/description of an asset)
* `ListAssets` (Lists assets)
* `GetAssetGroup` / `UpdateAssetGroup` / `DeleteAssetGroup` / `ListAssetGroups` (CRUD and listing of asset groups)
